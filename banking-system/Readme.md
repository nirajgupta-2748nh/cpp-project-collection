# banking system Overview

This is a simple menu-driven Banking Management System implemented in C++.
The program simulates basic banking operations such as checking balance, depositing money, and withdrawing funds using a secure PIN-based system.


# Features
 * PIN-based authentication for secure access
 * Check account balance
 * Deposit money into account
 * Withdraw money with balance validation
 * User-friendly menu-driven interface


# Concepts Used
* Object-Oriented Programming (OOP)
* Classes and Objects
* Encapsulation (private data members)
* Conditional statements (if-else)
* Loops (while, switch-case)


# How to Run
1. Compile the program :
g++ main.cpp -o bank
2. Execute the program :
./bank
