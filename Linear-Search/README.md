# Linear Search in C++

A simple C++ program that demonstrates the **Linear Search algorithm**. It takes user input for an array and searches for a specified element.

## Features

* User input for array elements
* Searches element using linear search
* Displays index if found, else shows not found

## Concepts Used

Arrays, loops, and conditional statements

## Usage

Compile and run the program, enter array values, and search for an element.
